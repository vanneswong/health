# 健康助手 (Health Buddy) - 开发计划

## 开发阶段

### Phase 1: 项目初始化与基础搭建

#### 任务清单
1. **创建项目目录结构**
   - 创建 frontend 和 backend 目录
   - 初始化 Git 仓库

2. **后端初始化**
   - 创建 Go module
   - 安装依赖包 (gin, jwt-go, bcrypt)
   - 设置 JSON 文件存储
   - 创建数据模型

3. **前端初始化**
   - 使用 Vite 创建 React 项目
   - 安装依赖包 (Material UI, Recharts, axios, react-router)
   - 配置项目基础结构

### Phase 2: 后端开发

#### 任务清单
1. **认证模块**
   - 实现用户登录接口
   - 实现 JWT Token 生成和验证
   - 实现密码修改接口
   - 创建认证中间件

2. **血压记录模块**
   - 实现 CRUD 接口
   - 实现记录列表查询（分页、排序、筛选）
   - 添加数据验证

3. **血糖记录模块**
   - 实现 CRUD 接口
   - 实现测量类型枚举
   - 添加数据验证

4. **用药管理模块**
   - 实现用药计划 CRUD
   - 实现服药打卡接口
   - 实现用药统计接口

5. **统计模块**
   - 实现血压趋势数据查询接口
   - 实现血压统计汇总接口
   - 实现血糖趋势数据查询接口
   - 实现血糖统计汇总接口（中国标准）

### Phase 3: 前端开发

#### 任务清单
1. **认证页面**
   - 登录页面
   - 登出功能
   - 修改密码页面

2. **个人资料页面**
   - 显示个人资料
   - 编辑个人资料

3. **血压记录页面**
   - 记录列表页面（表格展示）
   - 新增记录表单（对话框）
   - 编辑记录表单
   - 删除确认对话框
   - 记录详情展示

4. **血糖记录页面**
   - 记录列表页面（表格展示）
   - 新增记录表单（对话框）
   - 编辑记录表单
   - 删除确认对话框

5. **用药管理页面**
   - 用药计划列表
   - 新增/编辑用药计划
   - 服药打卡页面
   - 今日用药进度展示

6. **数据展示页面**
   - 血压趋势图表页面（折线图）
   - 血糖趋势图表页面（折线图）
   - 统计汇总页面（整合血压和血糖）
   - 健康提醒组件

7. **通用组件**
   - 导航栏/侧边栏
   - 响应式布局
   - 血压级别指示器
   - 血糖级别指示器

### Phase 4: 集成与测试

#### 任务清单
1. **前后端集成**
   - 连接前后端 API
   - 处理跨域问题
   - 测试完整流程

2. **功能测试**
   - 测试登录/登出流程
   - 测试血压记录 CRUD
   - 测试血糖记录 CRUD
   - 测试用药管理功能
   - 测试图表展示
   - 测试统计功能

3. **响应式测试**
   - PC端测试
   - 移动端测试

### Phase 5: 完善与提交

#### 任务清单
1. **完善细节**
   - 添加错误提示
   - 添加加载状态
   - 优化用户体验
   - 时间处理统一使用北京时间(UTC+8)

2. **创建 README**
   - 项目说明
   - 安装和运行指南

3. **Git提交**
   - 提交所有代码到本地仓库
