# 血压宝 (BP Buddy) - 需求文档

## 1. 项目概述

### 1.1 项目名称
血压宝 (BP Buddy)

### 1.2 项目简介
一个用于记录和追踪血压数据的Web应用，帮助用户便捷地记录血压测量结果，并通过图表和统计分析了解血压变化趋势。

### 1.3 技术栈
- **前端**: React 18 + Material UI + Recharts
- **后端**: Go (Gin框架)
- **数据存储**: JSON文件存储（替代SQLite，因Windows环境编译限制）
- **认证**: JWT Token

## 2. 功能需求

### 2.1 用户管理
- 单用户模式，需要登录验证
- 登录功能（用户名/密码）
- 登出功能
- 支持修改密码

### 2.2 血压记录管理

#### 2.2.1 记录字段
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 收缩压(高压) | 整数 | 是 | mmHg |
| 舒张压(低压) | 整数 | 是 | mmHg |
| 脉搏 | 整数 | 是 | 次/分钟 |
| 测量时间 | 日期时间 | 是 | 默认当前时间 |
| 用药 | 文本 | 否 | 记录是否服药及药物名称 |
| 备注 | 文本 | 否 | 其他说明 |

#### 2.2.2 操作功能
- 新增血压记录
- 查看记录列表（分页、排序）
- 编辑已有记录
- 删除记录（确认删除）
- 查看记录详情

### 2.3 数据展示

#### 2.3.1 图表趋势
- 血压趋势折线图（显示收缩压、舒张压、脉搏）
- 支持选择时间范围（近7天、近30天、近90天、全部）
- 鼠标悬停显示具体数值

#### 2.3.2 统计汇总
- 日/周/月平均值统计
- 最高值/最低值/正常范围
- 血压分布情况

#### 2.3.3 健康提醒
- 根据血压值给出健康建议
- 血压分级提示（正常/偏高/高血压）
- 异常值醒目标识

### 2.4 界面要求
- 响应式设计，支持PC和移动端
- 简洁美观的用户界面
- 良好的用户体验

## 3. 非功能需求

### 3.1 性能要求
- 页面加载时间 < 2秒
- API响应时间 < 500ms

### 3.2 安全要求
- 密码加密存储（bcrypt）
- JWT Token认证
- API请求保护

### 3.3 兼容性
- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 响应式布局支持移动端

## 4. 数据模型

### 4.1 数据存储
数据存储在 `bp_buddy.json` 文件中，包含用户和血压记录数据。

### 4.2 用户数据结构
```json
{
  "id": 1,
  "username": "admin",
  "password_hash": "...",
  "created_at": "2024-01-01T00:00:00Z",
  "updated_at": "2024-01-01T00:00:00Z"
}
```

### 4.3 血压记录数据结构
```json
{
  "id": 1,
  "user_id": 1,
  "systolic": 120,
  "diastolic": 80,
  "pulse": 72,
  "measured_at": "2024-01-01T08:00:00Z",
  "medication": "",
  "notes": "",
  "created_at": "2024-01-01T08:00:00Z",
  "updated_at": "2024-01-01T08:00:00Z"
}
```

## 5. API接口设计

### 5.1 认证接口
- `POST /api/auth/login` - 用户登录
- `POST /api/auth/logout` - 用户登出
- `PUT /api/auth/password` - 修改密码

### 5.2 血压记录接口
- `GET /api/records` - 获取记录列表
- `GET /api/records/:id` - 获取单条记录
- `POST /api/records` - 创建记录
- `PUT /api/records/:id` - 更新记录
- `DELETE /api/records/:id` - 删除记录

### 5.3 统计接口
- `GET /api/stats/summary` - 获取统计汇总
- `GET /api/stats/trend` - 获取趋势数据

## 7. 血压健康标准参考

| 分类 | 高压(mmHg) | 低压(mmHg) |
|------|-------------|-------------|
| 正常血压 | <120 | <80 |
| 正常高值 | 120-139 | 80-89 |
| 高血压1级(轻度) | 140-159 | 90-99 |
| 高血压2级(中度) | 160-179 | 100-109 |
| 高血压3级(重度) | ≥180 | ≥110 |
