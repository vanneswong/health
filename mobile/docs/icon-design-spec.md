# 血压宝应用图标设计规格

## 当前登录页Logo设计

登录页使用的logo是矢量图标组合，具体规格如下：

### 设计元素

| 元素 | 规格 |
|------|------|
| **背景圆形** | 直径88px，圆角44px |
| **背景颜色** | rgba(255, 255, 255, 0.2) - 半透明白色（登录页使用） |
| **应用图标建议背景** | #1976d2 - 主题蓝色 |
| **图标** | MaterialCommunityIcons `heart-pulse` |
| **图标颜色** | #ffffff - 白色 |
| **图标大小** | 48px（相对88px背景） |

### 颜色参考

| 颜色名称 | HEX值 | 用途 |
|----------|-------|------|
| 主题蓝 | #1976d2 | 背景、按钮 |
| 白色 | #ffffff | 图标、文字 |
| 半透明白 | rgba(255,255,255,0.2) | 登录页logo背景 |

### 应用图标尺寸要求（Expo）

| 图标类型 | 尺寸 | 文件路径 |
|----------|------|----------|
| 主图标 | 1024×1024px | `assets/icon.png` |
| Android自适应图标 | 1024×1024px | `assets/adaptive-icon.png` |
| 启动画面图标 | 1284×1284px | `assets/splash-icon.png` |

### 设计建议

推荐使用以下工具制作正式PNG图标：

1. **Figma** - 免费，专业设计工具
2. **Canva** - 简单易用，有模板
3. **IconKitchen** - 专门制作应用图标
4. **在线工具** - https://icon.kitchen 或 https://appicon.co

### 设计草图示意

```
┌─────────────────────┐
│    圆形背景          │
│   #1976d2 蓝色      │
│                     │
│      ♥─♥           │  ← 心跳图标
│     heart-pulse     │
│      #ffffff        │
│                     │
└─────────────────────┘
```

### 快速制作方法

在图标制作工具中：
1. 创建1024×1024画布
2. 填充背景色 #1976d2
3. 添加心跳/心形图标
4. 图标颜色 #ffffff
5. 导出PNG

完成后替换 `mobile/assets/` 目录下的图标文件即可。
